# Allen | Personal Portfolio

A modern personal portfolio website built with Next.js, TypeScript and Tailwind CSS.

## About

This project represents my personal journey, values, achievements, and ambitions as I prepare to pursue higher education abroad.

Rather than being just a portfolio, this website tells the story of who I am, what drives me, and where I am going.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion (planned)
